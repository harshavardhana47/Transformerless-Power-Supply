Transformerless Power Supply (5V Output) – KiCad Design

Designed a compact transformerless AC to DC power supply delivering regulated 5V output using KiCad, optimized for low-cost and space-constrained applications.

🔌 Implemented capacitive dropper circuit for voltage reduction
🔁 Used bridge rectifier (1N4007 diodes) for AC to DC conversion
⚡ Integrated filter capacitors for ripple minimization
🎯 Added Zener diode regulation + LM7805 for stable 5V output
💡 Included LED indicator for power status
✅ Verified using ERC (0 errors) ensuring schematic reliability
